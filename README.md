## AQA Courses - Lecture 2 - Demo

### Installation

Create a fork of a demo [repo](https://github.com/sskorol/aqa-heart-java-demo.git).

```shell
git clone https://github.com/[YOUR_GH_ID]/aqa-heart-java-demo.git && cd aqa-heart-java-demo
docker-compose up -d
```

### Running Tests

```shell
./gradlew clean test
```

### Report Generation

```shell
./generate_report.sh
```
